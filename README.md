# ⚙️ C++ Mastery Showcase

A curated collection of C++ programs and projects demonstrating deep command 
over the language — from fundamentals to advanced systems programming.

## 📌 What This Covers

- **Core Language** — Pointers, references, type system, operator overloading
- **OOP & Design Patterns** — Inheritance, polymorphism, RAII, SOLID principles
- **Modern C++ (C++11/14/17/20)** — Smart pointers, lambdas, move semantics, 
  concepts, ranges, and coroutines
- **STL & Templates** — Containers, iterators, algorithms, template metaprogramming
- **Memory Management** — Manual allocation, custom allocators, stack vs heap
- **Concurrency** — Threads, mutexes, condition variables, async/futures
- **Data Structures & Algorithms** — Hand-rolled implementations for deeper understanding
- **System-Level Programming** — File I/O, process handling, low-level optimizations

## 🎯 Purpose

This repository serves as a living portfolio of C++ proficiency — not just 
"it compiles," but code that is idiomatic, efficient, and well-reasoned. 
Each module is written to reflect real engineering judgment, not just syntax familiarity.

## 🛠️ Build & Run

\`\`\`bash
g++ -std=c++17 -O2 -Wall filename.cpp -o output
./output
\`\`\`
